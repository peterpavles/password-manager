Running Standalone JAR

Before running from JAR file make sure that Java Cryptography Extension (JCE) Unlimited Strength Jurisdiction Policy Files are properly installed.

java -jar password-manager-<version>.jar

Running Platform Specific Binary

Platform specific binaries are packages with JRE and don't require additional configuration.
